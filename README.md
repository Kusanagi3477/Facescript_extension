# Fa(c_e)Script

Fa(c_e)Script拡張機能はo-uki氏作の顔文字プログラミング言語「fa(c_e)script」を書きやすくするものです。
主に色付けやコード補完などをします。

## 変更履歴

・Fa(c_e)scriptのバージョン1.3.0系の構文に対応。

・⊂(¯^¯)⊃をseparateではなくandに変更。
